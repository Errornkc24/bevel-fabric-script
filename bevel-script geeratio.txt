  Key Features

  - Consensus selection: Asks Raft (Fabric 2.5.4) or BFT (Fabric 3.x) at start
  - Resumable: --resume flag picks up from last completed phase
  - Checklist display: Shows progress with checkmarks after every phase
  - Error handling: Retry/skip/abort on failure, log file at
  ~/.bevel-setup/setup.log
  - Dual sync: Auto-checks remote ports first, falls back to manual confirmation
  - Interactive: Asks for all user inputs (IPs, tokens, GitHub creds, passwords)

  ---
  How to Run Each Script

  Script 1 (Per-PC):

  # On EACH PC:
  cd claude-bevel-automation-script/scripts/
  ./1-per-pc-setup.sh

  Script 2 (Controller):

  # First set up SSH keys, then on controller only:
  ssh-copy-id user@PC1_IP && ssh-copy-id user@PC2_IP && ssh-copy-id user@PC3_IP
  ./2-controller-setup.sh

  Script 3 (Hybrid):

  # Step 1: On ALL 3 PCs:
  ./3-hybrid-setup.sh infra

  # Step 2: On CONTROLLER only (after all PCs finish infra):
  ./3-hybrid-setup.sh deploy

  # Anytime health check:
  ./3-hybrid-setup.sh health
